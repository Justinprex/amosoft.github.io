Deep Dive: STM32 + Xbox JTAG

The STM32 was interfaced with the Xbox JTAG to reverse engineer memory access and instruction hooks. Key steps included:

  • Dumping flash memory via SWD interface.
  • Disassembling boot ROM with Ghidra.
  • Bypassing signature checks for custom payload execution.

Result: Successful modification and deeper understanding of the console security layers.

Deep Dive: Firmware Analysis

This project focused on understanding a closed-source device firmware. Highlights included:

  • Extracting the filesystem with binwalk.
  • Identifying hardcoded keys inside the binary.
  • Tracing communication protocols with Wireshark.

Outcome: Security flaws were documented for responsible disclosure.
